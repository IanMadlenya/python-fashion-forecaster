About Python Fashion Forecaster

I created this app to gather social api data from the Internet about fashion into a central location, in order to find trends from a computational approach. It is still in beta.

Ingredients

This app is made with several open source software: Django Web framework for Python, MySQL database, and for visualizations: D3.js and Flotr2.js.
